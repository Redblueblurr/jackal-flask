Jackal Flask

## User Guide

### Clone this project

`git clone https://github.com/wgx731/jackal-flask.git`

### Swtich to tutorial branch

* `git checkout <branch-name>` **setup, mvc, rest, persistent, auth, final**
* follow instructions on `README.md`
